> [!NOTE]

> `%APPDATA%` is `C:\Users\{UserName}\AppData\Roaming` 
> `Window + R` and enter `%AppData%` to see exactly location.
> `%LOCALAPPDATA%` is `C:\Users\{UserName}\AppData\local` 
> `Window + R` and enter `%LOCALAPPDATA%` to see exactly location.
